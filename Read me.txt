feature of project

seller side:
1. Each individual seller can register his own profile by sign up
2. Seller can create/view/update/delete his product from his own database by login his own account
3 .Seller can edit his profile



customer  side:
1. Each individual Customer can register his own profile by sign up
2. Customer can select/remove product from cart
3 Customer can edit his profile
4.Without login any account customer can select product.but only after  login he can purchase it.

cart:
1.It shows no of items selected as well as total amount.

How to run the Online Shopping Portal using PHP and MySQL
---------------------------------------------------------------------------------------------------
1. Download the  zip file

2. Extract the file and copy shopping folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/a1)

5. Database and database  tables is created automatically once the file run on browser
